# docker
Docker Compose scripts, intended for use in Portainer Stacks.

## Folders

### AI
GPU-accelerated AI services stack running on the local machine.

- **Ollama** - Local LLM inference
- **Open WebUI** - Chat interface for Ollama with image generation, TTS, and STT
- **ComfyUI** - Image generation via Flux
- **Kokoro** - Text-to-speech
- **WhisperX** - Speech-to-text
- **SearXNG** - Local web search for RAG

### Applications
General self-hosted applications and services.

- Flame - Homepage/dashboard
- IT Tools - Collection of developer/IT utilities
- Ntfy - Notification service
- Nginx Proxy Manager - Ingress and reverse proxy
- Picoshare - Secure file and text sharing
- Watchtower - Automatic container updates
- Wireguard VPN (wg-easy)

### Games
Self-hosted game servers.

- EmulatorJS - Browser-based emulator for ROMs
- Minecraft - Bedrock Edition survival server
- Veloren - Open source MMORPG

### Untested
Compose files not yet in production.
